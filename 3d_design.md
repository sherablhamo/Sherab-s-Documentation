# 🖨️ 3D Designing

## What is 3D Design?
The process of using computer software to create objects in three dimensions (length, width, and height) is known as 3D design. Unlike 2D drawings, 3D designs appear realistic and can be viewed from multiple angles.

These designs are used in:
- 3D printing  
- Games  
- Engineering  
- Animation  
- Architecture  

![3D Design](https://drive.google.com/thumbnail?id=1_tDN3aQQ4jegLLmSNa0Z8AUxAOmyQK1A&sz=w800)

Using 3D design, we can create digital models of real or imaginary objects.

**Example:**  
You can combine shapes like a cube, sphere, or cylinder to create a robot or a cup.

![Example](https://drive.google.com/thumbnail?id=1K0Ioo2ix7Nip9_eooqN1RtbEIFqIZ1oz&sz=w800)

---

## Why is 3D Design Important?

- Helps visualize ideas clearly  
- Used in engineering and product design  
- Important for 3D printing  
- Improves creativity and problem-solving  
- Used in games and animation  

---

## Software Used for 3D Design

1. Tinkercad – Beginner-friendly

# CAD (Computer-Aided Design)

CAD stands for Computer-Aided Design. It is used to create and modify 2D and 3D models using computers.

![CAD](https://drive.google.com/thumbnail?id=1jCHDxn7L6P49GtJuYIDK4wc0CH_I1tip&sz=w800)

CAD helps engineers and designers create accurate models before building real products.

### Dimensions used in CAD:
- Length  
- Width  
- Height  
- Angles  

---

## Why CAD is Important

- Makes designs accurate  
- Saves time  
- Easy to edit  
- Used in engineering and architecture  
- Helps prepare models for 3D printing

# Additive vs Subtractive Manufacturing

## Additive Manufacturing
Material is added layer by layer.

**Example:** 3D Printing

![Additive](https://drive.google.com/thumbnail?id=1BLae4-dHC0cuHFlDU-nvQLi2R47kwgXo&sz=w800)

---

## Subtractive Manufacturing
Material is removed from a solid block.

**Examples:**
- CNC Milling  
- Laser Cutting  

![CNC](https://drive.google.com/thumbnail?id=1tI8cBYbILZuB_Rni9N2HZi-WdBgC1ppM&sz=w800)

![Laser](https://drive.google.com/thumbnail?id=1snDIGQ7cSaz1YBlwnCRB74g3QFAAj3L0&sz=w800)

---

## Comparison

| Additive | Subtractive |
|----------|------------|
| Material added | Material removed |
| Less waste | More waste |
| Used in 3D printing | Used in CNC |
| Complex shapes | High precision |

# Parametric Design

## What is it?
In parametric design, models depend on values like length, width, or angles.  
If one value changes, the whole design updates automatically.

---

## Steps:
1. Create sketch  
2. Add dimensions  
3. Convert to 3D  
4. Change parameters  
5. Design updates automatically  

---

## What is Kerf?
Kerf is the small amount of material removed during cutting.  
It must be considered for proper fitting.

---

## File Types

- **STL** → Shape of model  
- **OBJ** → Shape + color  
- **G-code** → Instructions for printer
