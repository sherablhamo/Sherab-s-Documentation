# **Laser Cutting**

## What is Laser Cutting?
Laser cutting is the process of using a laser beam to cut or engrave materials such as wood, glass, cardboard, or plastic. It is widely used in production, prototyping, and design because it allows precise shapes and detailed patterns.

## How to Do Laser Cutting?

### 1. Choosing the Design
I used Inkscape to create the 2D design. First, I searched Pinterest for a reference image (“Giraffe black and white”) and imported it into Inkscape to use as a guide for tracing.

<img src="https://drive.google.com/thumbnail?id=1T3gfKV3s13QLFW4PN4QddrP2NgkxoV46&sz=w800" alt="Design Reference">

---

### 2. Converting Image to Vector (Trace Bitmap)
Imported images may have low resolution or rough edges. To fix this, I used **Trace Bitmap**.

Steps:
- Go to Path → Trace Bitmap  
- Or right-click → Trace Bitmap  

This converts the image into a clean vector outline suitable for laser cutting.

<img src="https://drive.google.com/thumbnail?id=1PtSohm2w-4DbJ50hfUrkjOfJT71cXyiF&sz=w800" alt="Trace Bitmap">

---

### 3. Adjust Settings
- Select **Brightness Cutoff**
- Adjust until the outline becomes clear and clean

<img src="https://drive.google.com/thumbnail?id=1_jK0N98UTPUZiKJAHIHZU-7Ce997btn8&sz=w800" alt="Adjust Settings">

---

### 4. Apply and Clean Design
- Click **Apply**
- Use the selection tool (top left cursor)
- Drag to create a clean vector copy
- Delete the original noisy image

<img src="https://drive.google.com/thumbnail?id=1yf-huuqlWjPUWOyRTqszoPdVbx6f9DSH&sz=w800" alt="Clean Vector">

---

### 5. Create Cutting Boundary
- Use the rectangle tool
- Draw a boundary around the design  
This defines the cutting area.

<img src="https://drive.google.com/thumbnail?id=1YIl97CX4Jzv7W8YX5GAAsVu8Sf2DnfTY&sz=w800" alt="Cut Boundary">

---

### 6. Laser Settings
Adjust settings in the laser software:

- Speed  
- Power  
- Mode: Engraving / Vector cutting  

Select **Vector + Deep Engraving** if required.

# Steps for Laser Cutting (Design Process in Inkscape)

## 1. Choose a Design
- Search for a design (e.g., on Pinterest)  
- Example: “Giraffe black and white”  
- Copy and paste the image into Inkscape  
- Use it as a reference to trace  

---

## 2. Convert Image to Vector (Trace Bitmap)
Imported images may have:
- Low resolution  
- Blurry edges  
- Small unwanted spots  

To fix this:
- Go to **Path → Trace Bitmap**  
- OR right-click → Trace Bitmap  

This converts the image into a clean vector outline.

---

## 3. Adjust the Settings
- Use **Brightness Cutoff**  
- Adjust until:
  - The outline becomes clear  
  - Unwanted details are removed  

---

## 4. Apply and Clean the Design
- Click **Apply**  
- Use the cursor tool (top left)  
- Drag the image  
- A clean duplicate appears  
- Delete the original noisy image  

---

## 5. Create Cutting Boundary
- Select the **Rectangle tool**  
- Draw a box around your design  
- This defines the cutting area  

---

## 6. Set Laser Cutting Parameters
Go to engrave / laser settings and adjust:

- Speed  
- Power  
- Mode:
  - Engraving (surface)
  - Vector cutting (deep cutting)  

You can choose:
- Vector + Deep Engraving (if needed)

---

## 7. Send to Machine (Print)
- Click **Print**  
- Ensure correct settings before sending  

---

# Machine Operation Steps

## Before Starting
Turn on:
- Air exhaust system (removes smoke)  
- Air pump (prevents burning and improves cutting)  

---

## Machine Setup
Adjust:
- Speed  
- Power  
- Auto Focus  

Place material properly on the bed.

---

## Start Cutting
- Press the **Start / Button**  
- The laser machine begins cutting  

---

## After Cutting
- Wait until the machine stops completely  
- Remove material carefully  
- Check:
  - Clean cuts  
  - No burns or errors
